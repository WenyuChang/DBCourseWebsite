Oracle与数据库管理  实验指导书

前  言

Oracle Database 10g是一种具有网格计算机架构的数据库系统，在性能、可扩展性、安全性、可管理性和系统可用性等方面具有绝对的领先优势。本课程全面讲述了Oracle Database 10g的特点和功能。从安装配置、安全性、可用性、互操作性、PL/SQL、可开发性、商业智能等多个方面，对Oracle Database 10g的各项技术进行由浅入深的讲述。

学习本课程，通过学习Oracle数据库安装、管理及开发。为以后希望从事Oracle数据库维护、开发打下课程。。

一、实验目的

      更好的理解Oracle数据库维护、开发。

二、实验要求

1、每次实验前学生必须根据实验内容认真准备。

2、在指导教师的帮助下能够完成实验内容。

3、实验结束后总结实验内容、书写实验报告。

4、遵守实验室规章制度、不缺席。

5、实验学时内必须做数据结构的有关内容，不允许上网聊天或玩游戏。

三、实验环境： Oracle 10g

四、说明

   Oracle与数据库管理是本校计算机有关专业特色课程。  

五、实验报告的书写要求

1. 明确实验的目的及要求;

2. 记录实验的输入数据和输出结果;

3. 说明实验中出现的问题和解决过程;

4. 写出实验的体会和实验过程中没能解决的问题;

六、参考书目

        《Oracle Database 10g基础教程》 闪四清著 清华大学出版社
